## Networking Fundamentals Explorer

An interactive, password-protected web application for exploring a 6-day networking fundamentals curriculum, created for a Government sponsored, non- credit, College Program.

This project is a single-page application (SPA) that acts as an interactive hub for a networking fundamentals course. It features a two-column layout where users can navigate between a course overview and 6 distinct "days" of content.

The application is password-protected and uses JavaScript to dynamically render content, including custom HTML/CSS visualizations and a Chart.js-powered bar chart.
